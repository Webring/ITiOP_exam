# ПРОЧТИ МЕНЯ

Все решения проверены в Clion, и могут не компилироваться в других IDE (ну тут сами виноваты, нужно нормальным софтом пользоваться).

Решения прокоментированы более менее (чаще, конечно, прокоментированы чисто для галочки). Также к решениям приложены тестовые файлы.

**Важно!** Имя тестовых файлов вынесены в вверх файла в директивы препроцессора. Примерно так:
> #define INPUT_FILE_PATH "../task N/test.txt"

Если вы используете Clion, то перед именем файла нужно добавлять "../" (вообще адрес файла указывается относительно директории cmake-build-debug, а не
файла с исходниками, и "../" говорит о том, что файл лежит в родительской директории).
Что и как там в visual studio понятия не имею, но вроде там все таки путь к файлу записывается относительно файла с исходниками (Ну те без "../").
